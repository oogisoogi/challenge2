콘서트 예약 플랫폼을 만들 것입니다.

5 page로 구현합니다.

로그인/회원가입 인증기능은 구현하지 않습니다. 모두 비로그인으로 사용가능합니다.

## 페이지

1. 홈(콘서트 목록): 예약가능한 콘서트목록이 표시됩니다. 진행일 전날까지만 예약가능하며, 예약인원/정원이 표시됩니다.
2. 콘서트 상세: 콘서트 상세 정보가 표시되며, 예약하기 버튼을 클릭해 예약 프로세스를 시작할 수 있습니다.
3. 콘서트 예약: 
    - 해당 콘서트의 빈자리가 바둑판 형태로 표시되며, 가로4x세로20칸 구역이 4번 반복됩니다.
    - 각 칸은 비어있거나 예약된 상태를 표시합니다.
    - 비어있는 칸을 클릭 시 해당 칸이 예약대상 상태가 됩니다.
    - 한번에 최대 4칸까지 예약가능합니다.
    - 우측사이드바에 예매가능한 남은좌석수가 표시되며, 선택된 예약대상 좌석이 있는 경우 각 좌석의 정보도 표시됩니다.
    - X 아이콘이 표시되며 클릭시 해당 좌석이 예약대상 상태가 해제됩니다.
    - 1칸 이상 선택한 뒤에 예약하기 버튼을 누르면 '정보입력 단계로 전환됩니다.
    - '정보입력 단계에서는 예약자명, 휴대폰번호, 비밀번호4자리를 입력받고, 제출 시 다시한번 각 좌석들의 예약상태를 점검합니다.
    - 하나라도 예약완료된 좌석이 있다면 실패한 뒤 이미 예약된 좌석입니다. alert를 보여줍니다.
    - 성공했다면 예약완료 페이지로 이동합니다.
4. 예약 완료: 예약이 완료되면 예약완료 페이지로 이동합니다. 해당 예약건에서 예약된 좌석들의 정보가 표시되며, 예약취소 버튼을 클릭해 예약을 취소할 수 있습니다.
5. 예약 조회: 휴대폰번호와 비밀번호4자리를 입력하면, 해당 정보로 예약된 건들이 표시됩니다. 예약취소 버튼을 클릭해 예약을 취소할 수 있습니다.
